Machine learning algorithm to predict house prices in Ames, Iowa based on 79 variables. This is my second capstone project for Harvardx's data science professional certificate program.

Directories:  
 - report -> Different file types for the project final report. 
 - datasets -> csv files for datasets used and description txt file. 
 - scripts -> R scripts used when creating models. 
 - predictions -> csv files for predictions of various models. 
 - rds -> rds files used when creating models to save time loading objects previously created.
